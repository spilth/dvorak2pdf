# dvorak2pdf

Toying around with generating a [Dvoark](http://www.dvorakgame.co.uk/) deck from a YAML file.

## Usage

- Create cards in `input/cards`.
- Run `ruby cards.rb`
- Open `output/cards.pdf`.
